# Code Review Instructions

When I ask you to review, analyze, or summarize a script, please follow the comprehensive review framework defined in `/prompts/script-review-prompt.xml`.

## Usage Examples

- "Review this script" - Apply the full framework to the current file
- "Analyze script.py" - Apply the full framework to the specified file
- "Summarize this code" - Apply the full framework

## Framework Location

The complete review framework is stored in: `./prompts/script-review-prompt.xml`

Please read and follow all sections in order, providing a complete markdown document as output.

## Output Format

Generate a comprehensive markdown review document following all sections defined in the XML framework, including:
1. High-level summary
2. Statistics
3. Quality metrics
4. Dependencies
5. Architecture analysis
6. Performance analysis
7. Security assessment
8. Flow diagram
9. Maintainability indicators
10. Detailed component descriptions
11. Executive summary with recommendations
